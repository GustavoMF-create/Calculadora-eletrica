# 🔌 Calculadora de Consumo de Energia

Este projeto é uma **calculadora simples de consumo de energia elétrica** desenvolvida em **JavaScript**. O programa permite que o usuário informe o nome de um eletrodoméstico, sua potência e o tempo de uso diário para calcular o **consumo mensal em kWh** e uma **estimativa do custo na conta de energia**.

A aplicação aceita valores de potência em **Watts (W)** ou **Quilowatts (kW)**. Caso o usuário informe a potência em Watts, o programa faz automaticamente a conversão para **quilowatt-hora (kWh)** antes de realizar o cálculo mensal. Em seguida, o consumo é multiplicado por um valor médio de **R$0,90 por kWh** para estimar o custo final.

Este projeto foi desenvolvido com o objetivo de **praticar lógica de programação**, trabalhando com entrada de dados (`prompt`), estruturas condicionais (`if/else`), cálculos matemáticos e formatação de números em **JavaScript**.

Ele também ajuda a demonstrar de forma prática **como estimar o consumo de energia de aparelhos domésticos**, permitindo entender melhor o impacto do uso diário de eletrodomésticos na conta de luz.
